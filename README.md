# 📝 웹 게시물 시스템

간단하고 직관적인 **웹 기반 게시물 관리 시스템**입니다.

## ✨ 주요 기능
- ✅ 게시물 작성 (제목 + 내용)
- ✅ 게시물 목록 보기
- ✅ 게시물 삭제
- ✅ 로컬 스토리지에 자동 저장 (브라우저를 닫아도 유지)
- ✅ 모바일 친화적 반응형 디자인

## 🚀 바로 사용하기

**GitHub Pages 링크**: [https://jhh78.github.io/web-post-system/](https://jhh78.github.io/web-post-system/)

(아직 Pages 설정 안 하셨다면 아래 방법으로 설정하세요)

## 📋 GitHub Pages 배포 방법

1. Repository → **Settings** → **Pages**
2. **Source**에서 `Deploy from a branch` 선택
3. Branch를 `main` , 폴더를 `/ (root)` 로 선택
4. **Save** 클릭
5. 잠시 후 https://jhh78.github.io/web-post-system/ 로 접속

## 🛠 기술 스택
- HTML5 / CSS3
- Vanilla JavaScript
- LocalStorage (데이터 영속성)
